INSTRUCTIONS D’INSTALLATION PWA POUR e-META

1. Placer les fichiers suivants dans le dépôt GitHub Pages :
- index.html
- style.css
- manifest.json
- service-worker.js
- favicon.ico
- Official-eMETA-logo.png

2. Vérifier que le fichier index.html appelle :
- le manifest.json via <link rel="manifest" ...>
- le favicon.ico
- le service-worker.js via un <script> (en bas de page)

3. Activer GitHub Pages depuis les paramètres du dépôt.

4. Tester l’installation PWA :
- Ouvrir le site avec Chrome/Edge
- Cliquer sur “Installer e-META” dans la barre d’adresse si disponible
